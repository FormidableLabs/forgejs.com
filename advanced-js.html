Advanced JavaScript Intensive Weekend

Every significant web application today has major components written in JavaScript. Moving from jQuery and DOM manipulation to full scale application development has a steep learning curve. We're here to help.

Over the course of one evening and two days, you will learn skills and concepts foundational to building full stack JavaScript applications under the guidance of world class mentors. The focus of the weekend will be hands on, project-based learning designed to prepare you to work independently afterwards.

Class size is limited to 24 attendees.

Am I prepared for the weekend?

You will be best served if you have a base level of understanding of the JavaScript language. At a minimum, this means fluency with:

  • Data types
  • Conditionals
  • Functions
  • Objects, properties & methods
  • Iterators
  • jQuery API for DOM manipulation

Course Outline

The core of the Advanced JavaScript weekend is understanding critical design patterns that enable largescale JavaScript applications.

FUNCTIONS AS FIRST CLASS TYPES
  • Functions as objects
  • Functions returning new functions
  • Functions with variable length arguments
  • Dynamic method calls
  • Lexical scope
  • Setting the context of the 'this' keyword (binding) with apply()
  • Currying with call()
CLASSES AND INHERITANCE
  • DIY/Rolling your own!
  • Backbone's way
ASYNC
  • Overview of async
  • Promises
  • Async lib
UNIT TESTING
  • Sinon
  • Mocha
