Am I prepared for the weekend?

You will be best served if you have a base level of experience with the JavaScript language. At a minimum, this means fluency with:

  • Data types
  • Conditionals
  • Functions
  • Objects, properties & methods
  • Iterators
  • jQuery API for DOM manipulation

If you aren't already at this level or are not sure what some of the above terms mean, we would recommend waiting to attend a ForgeJS event and starting with some of these online resources:

  • A re-introduction to JavaScript
  • Codecademy
  • Code School

Course Outline

The core of the Advanced JavaScript weekend is understanding critical design patterns that enable large scale JavaScript applications. The course proceeds with a series of projects into which the concepts and skills are integrated. The core topics are:

Higher Order JavaScript

We will ensure that you've mastered scoping, binding, understand how to treat functions as objects and generate new functions from within a function. We will also cover several of the ways that classes and inheritence have been implemented in JavaScript and ensure that you can take full advantage of it's prototypal nature regardless of what framework or platform you work within.

Mastering Asynchronous JavaScript

Working with asynchronous code can quickly become a mind bending excercise. You'll learn how to best use jQuery's Deffered and promises, and how to asynchronously process lists with the async library.

Event Driven Architecture

Regardless of what framework or platform you work with, events are the glue that allow your code to scale. Move beyond basic DOM events with an overview of custom DOM events, and event APIs as implemented by several popular frameworks. You'll learn how and when to use your own non DOM events to allow your objects and classes to communicate with each other.

Best Practices

All ForgeJS courses have a review of, and emphasis on utilizing the best practices for version control and unit testing. In addition you'll have the opportunity to collaborate with small teams on your projects if you choose, but are welcome to work on your own. You are also welcome to work through your own real world projects with our mentors.
