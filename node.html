Course Outline

The course proceeds with a series of projects into which the concepts and skills are integrated. The core topics covered are:

Building Webservers with Node

Node excels at network communication and building web servers of all types. You'll learn how to spawn HTTP requests from within node, and build both traditional webservers and pure API servers that utilize both regular SQL and NoSQL (Redis, MongoDB, etc) to power your application.

Going Realtime with Socket.io

Node was built for async and you'll learn how to take maximum advantage of it to pipe data to your applications in real time. Two way communication gets complicated in a hurry and we'll make sure you don't get lost.

Workflow Automation with NPM and Grunt

Node isn't just for webservers, it's increasingly an invaluable part of managing your workflow and building your projects, even if the finished product doesn't involve Node at all. You'll learn understand how to spawn and control new processes from within Node and can configure your project to best take advantage of the thousands of available Node modules. In addition we'll provide a plethora of use cases for Grunt and ensure you can weave together your own recipes from the hundreds of available plugins.
